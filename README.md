## Getting Started
* [Node.js v16 (and npm v8) or latest version](https://nodejs.org/en/download/)

## Instructions
* Run program
```sh
npm install
```
```sh
npm start
```
* Access console page
```console
Login
```
```console
Click wallet icon
```
* Update account slots
```console
Enter command in console
```
```sh
40 cash
```
